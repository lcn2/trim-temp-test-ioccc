## To build:

```sh
    make all
```


## To use:

```sh
    ./applin
```


## Try:

```sh
    ./applin.c
```


## Judges' remarks:

The judges were so amused by this little program that we created an
award just for it.  This program is portable to the C and Fortran 77
compilers as well as executable by the Bourne shell!
