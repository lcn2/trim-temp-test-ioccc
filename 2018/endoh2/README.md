# To build:

```sh
make
```

### To run:

```sh
./prog
```

### Try:

```sh
./prog | tee prog_next.c
make prog_next
./prog_next | tee prog_next.c
make prog_next
./prog_next | tee prog_next.c

make python

make python3
```
