## To build:

``` <!---sh-->
    make all
```


## To use:

``` <!---sh-->
    ./ant Makefile

    ./ant Makefile rule
```

where `Makefile` is a Makefile to use and `rule` is an optional rule to use.


## Try:

``` <!---sh-->
    ./try.sh
```
